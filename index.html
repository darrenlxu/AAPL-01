Filtering of data

Now using MySQL we will import data into "project" schema and will rename the table to AAPL. 

Next we shall start to filter through the data.

Viewing AAPL dataset, 

 SELECT * FROM AAPL;

 We now have seen the entire dataset. The results have been returned in a table.

 

Summing data

SELECT SUM(Volume) FROM AAPL;

We have summed every AAPL stock on the market ~ 182,756,677,352 shares!

 

Finding the number of results of a specific case

SELECT COUNT(*) AS thecount FROM AAPL WHERE Volume > 200000000;

We can now see the number of results in our dataset named as "the count" where the volume on the day was greater than 200,000,000.
